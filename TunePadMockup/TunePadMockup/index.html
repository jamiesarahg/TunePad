Coding Key

Here are some of the things you need to know to code the nodes

  • node
    • Represents the node that you are editing in the block.
    • node is already defined for you
  • makeSound(note);
    • plays the note indicated.
    • Note choices are: 'A3', 'B3', 'C4', 'D4', 'E4', 'F4', 'G4'
    • Note must be a string
  • nodeTups
    • array of lists, each list represents one node on the canvas
    • first item of an internal list is the fabric node itself, second is its color
  • emit(node, endNodeTup)
    • Emits a emission from the node reprsented by the first argument to the node in the Tuple of the second argument
    • first argument is the node itself
    • Second argument is the tuple from nodeTupes
